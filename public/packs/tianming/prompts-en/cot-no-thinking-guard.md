## Output Format Restriction (Thinking Tags Prohibited)

**The following tags are strictly forbidden in your output:**
- ❌ `<thinking>` or any variant (`<think>`, `<reasoning>`, `<thought>`)
- ❌ Any content wrapped in thinking-type XML tags
- ❌ `<正文>` tags (output JSON directly)

Your output must be a **pure JSON object** — no reasoning blocks, no XML tags, no leading text, no trailing explanations. Start directly with `{`.
