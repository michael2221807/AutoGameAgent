# 世界详情描述档生成

你的任务是为一个游戏世界生成详尽的世界详情描述档。
这不是在写故事——你在 setup 世界的基础设定。

## 创角信息

```json
{{WORLD_SELECTION}}
```

## 主角人设

{{PLAYER_PROFILE}}

## 生成要求

生成一份 500-1500 字的世界详情描述档，包含：

1. **世界基调**：氛围、风格、主旋律
2. **核心规则**：力量体系、社会结构、经济形态
3. **历史概要**：关键历史事件、时代背景
4. **文化特征**：民俗、信仰、生活方式
5. **地理概述**：大陆/区域布局、气候特征

## 输出格式（严格遵守）

- **纯文本**，直接输出描述档内容
- 不需要 JSON 或 commands
- 禁止 `<thinking>` 标签
- 禁止代码围栏 ` ``` `

直接输出描述档正文。
