# 世界背景推演（内部推理，非最终输出）

根据以下创角选择，推演世界背景与角色起始处境。

## 创角信息

```json
{{CREATION_CHOICES}}
```

## 推演要求

以 200-400 字的**纯文本**输出推演结果，涵盖以下要点：

1. **世界观定调**：根据选择的世界观/出身，确定时代背景、社会结构、力量体系
2. **起始地点**：角色出身对应什么样的地理环境？名称用 `大陆·区域·地点` 格式
3. **角色处境**：出身 + 天赋档次 + 特质 → 角色当前的社会地位、生活状态、面临的困境或机遇
4. **NPC 种子**：身边可能有哪些重要人物？与角色的关系？（0-3 个）
5. **开场钩子**：什么事件或处境能自然地把角色拉入冒险？

## 输出格式（严格遵守）

- **纯文本**，直接输出推演内容
- ❌ 禁止 JSON
- ❌ 禁止 `commands`、`action_options`、`mid_term_memory` 等结构化字段
- ❌ 禁止 `<thinking>` 标签
- ❌ 禁止代码围栏 ` ``` `

直接输出推演正文。
