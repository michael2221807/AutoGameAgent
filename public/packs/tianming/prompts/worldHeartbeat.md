**你的核心任务**：为下列每个 NPC **代入该角色视角**，更新其 位置、内心想法、在做事项；必要时也可更新 外貌描述 / 身材描写 / 衣着风格（仅在发生长期变化时）。

**视角要求**：对每个 NPC，你必须站在**该 NPC 的立场**写出其「内心想法」与「在做事项」——用该角色的认知、动机与口吻，一句话描述他/她此刻的内心想法与正在做的事。禁止上帝视角或玩家视角；不写剧情概括，只写该角色自己的状态。

## 输出格式（仅输出一个 JSON 对象）

只输出一个 JSON，不要解释、思维链或代码围栏外的文字：

```json
{
  "commands": [
    {"action": "set", "path": "社交.关系[名称=李明阳].位置", "value": "青云镇·酒馆"},
    {"action": "set", "path": "社交.关系[名称=李明阳].内心想法", "value": "简短一句"},
    {"action": "set", "path": "社交.关系[名称=李明阳].在做事项", "value": "简短一句"}
  ]
}
```

### 约束

- 只允许 `set` 以下路径（且必须对应候选 NPC 名称）：
  - `社交.关系[名称=NPC名].位置`（value 为 string，具体地点名称）
  - `社交.关系[名称=NPC名].内心想法`（value 为 string，一句话）
  - `社交.关系[名称=NPC名].在做事项`（value 为 string，简短一句）
  - `社交.关系[名称=NPC名].外貌描述`（value 为 string；**仅在发生长期外貌变化时** —— 如衰老、伤疤、剃发、染发、刺青、长胖长瘦等）
  - `社交.关系[名称=NPC名].身材描写`（value 为 string；**仅在身材长期变化时** —— 如修炼突破、怀孕、受伤残、长期训练导致肌肉/体态改变）
  - `社交.关系[名称=NPC名].衣着风格`（value 为 string；**仅在日常衣着偏好转变时** —— 如身份改变、阶层变迁、季节转换等）
- 注意：使用 `path` 字段（不是 `key`），格式为 `社交.关系[名称=NPC名].字段名`
- 每个 NPC 可只更新部分字段，不必全部 set；不变化的可不输出
- **外貌描述 / 身材描写 / 衣着风格 默认不改动**：这 3 个字段代表 NPC 的长期视觉档案，生图流程依赖其稳定性。只有当剧情/时间跨度足以让外观真正改变时才更新，不要每回合都刷新。
- 禁止 set 其他路径（如 好感度、记忆、描述、性格特征 等）
- **位置描述**须为**具体**地点名（与已知地点列表一致或同级的具体名称），禁止「某城市」「某地点」「国外」等泛指
- 若某 NPC 无需变化可不输出；不要编造候选列表外的 NPC
