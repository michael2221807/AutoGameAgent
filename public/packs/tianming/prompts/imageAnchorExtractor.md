# 角色锚点提取

你是角色视觉锚点提取专家。任务：从角色资料中提取稳定的视觉身份标签（visual DNA），让同一角色多次生图保持一致。

## 角色资料

**角色名**: {{CHARACTER_NAME}}
**完整描述**: {{CHARACTER_DESCRIPTION}}
**外貌详情**: {{CHARACTER_APPEARANCE}}
**服饰**: {{CHARACTER_OUTFIT}}
{{ADDITIONAL_CONTEXT}}
{{EXTRA_EXTRACTION_REQUIREMENTS}}

## 提取规则

请从角色资料中提取以下维度的稳定视觉特征：

1. **外貌特征**: 发色、发型、瞳色、脸型、体型、肤色、年龄感
2. **胸部特征**: 胸型、大小（如角色为女性且有描述）
3. **发型**: 具体发型、发饰、长度、质感
4. **服饰基础**: 常驻衣着的类型、颜色、材质、层次、标志性配饰
5. **特殊标记**: 疤痕、胎记、刺青、必戴饰品
6. **年龄定位**: 具体年龄感或年龄段

### 排除项
- 临时状态（伤势、表情、当前动作）
- 场景依赖元素（光线、背景）
- 一次性服装变化

## 输出格式

请输出 JSON 结构（不包含代码块标记）：

```json
{
  "name": "锚点名称",
  "positive": "anchor_tag1, anchor_tag2, anchor_tag3, ...",
  "negative": "avoid_tag1, avoid_tag2, ...",
  "structuredFeatures": {
    "外貌": "...",
    "体型": "...",
    "胸型": "...",
    "发型": "...",
    "发色": "...",
    "瞳色": "...",
    "肤色": "...",
    "年龄": "...",
    "服饰基础": "...",
    "特殊标记": "..."
  }
}
```

正面提示词使用英文 tags，8-20 个，最重要的排前面。
结构化特征使用中文描述，每项简洁（5-15字）。
