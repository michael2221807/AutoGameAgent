## 输出格式（严格 JSON，禁止任何非 JSON 内容）

地点是**扁平数组**，通过 `上级` 字段引用父地点的 `名称` 来建树。
`名称` 使用 `·` 分隔的全路径格式。

```json
{
  "locations": [
    { "名称": "东华市", "描述": "繁华的现代都市", "类型": "城市", "上级": "" },
    { "名称": "东华市·静安区", "描述": "市中心核心区域", "类型": "区域", "上级": "东华市" },
    { "名称": "东华市·静安区·第七中学", "描述": "主角就读的公立中学", "类型": "建筑", "上级": "东华市·静安区" },
    { "名称": "东华市·静安区·第七中学·教室", "描述": "高二三班的教室", "类型": "建筑", "上级": "东华市·静安区·第七中学" }
  ],
  "inventory": [
    { "名称": "旧书包", "数量": 1, "描述": "用了两年的帆布书包", "类型": "装备" }
  ]
}
```

关键约束：
- `名称` 必须是 `·` 分隔的全路径（如 `东华市·静安区·第七中学`）
- `上级` 必须引用数组中另一个地点的 `名称`，顶级地点用空字符串 `""`
- 每个子地点的父地点必须也在数组中（先有父再有子）
- `类型` 使用自然语言标签（"城市"/"区域"/"建筑"/"自然"/"房间"等）
- 禁止 `<thinking>` 标签
- 只输出 JSON，不要输出任何解释文字
