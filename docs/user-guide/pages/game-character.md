# 角色详情 (Character Details)

> → [src/ui/components/panels/CharacterDetailsPanel.vue](../../../src/ui/components/panels/CharacterDetailsPanel.vue)
>
> 游戏内角色面板。路由 `/game`，Tab `character`。
> 展示主角全量属性：基础信息、先天/后天六维、社交关系摘要、身体档案（NSFW）、主角生图与锚点管理。
> 数据来自引擎状态树 `角色.*` 路径，所有字段通过 `useGameState()` 响应式读取。

## Tab: 基础

→ 模板 `activeTab === 'basic'`

### 基础信息卡片

| 字段 | 路径 | 内联编辑 |
|------|------|---------|
| 姓名 | `角色.基础信息.姓名` | 双击 → 文本输入，Enter 保存 / Esc 取消 / blur 保存 |
| 年龄 | `角色.基础信息.年龄` | 双击 → 数字输入，同上 |
| 性别 | `角色.基础信息.性别` | 点击 → select 下拉选择（男/女/其他），通过 `charEditor.updateField()` 保存（Story 2） |
| 职业/地位 | `角色.基础信息.职业` | 点击 → 内联文本输入，blur/Enter 保存（Story 2） |
| 当前位置 | `角色.位置` | 点击 → select 下拉选择（从 `世界.地点信息` 读取选项），通过 `charEditor.updateField()` 保存（Story 2） |

内联编辑机制：双击行 → 行内出现 `<input>`，键入新值 → Enter/blur 提交。保存后 Toast "已更新"。

**三个 tier section：**

## Tab: 属性

→ 模板 `activeTab === 'attributes'`

### 先天六维（基线 1-10）

仅在 `角色.身份.先天六维` 对象非空时显示。

```
┌─ 先天六维  基线 · 1-10 ──────────────────────────────┐
│  体质   7 / 10   [██████████████████████░░░░░]       │
│  灵力   5 / 10   [████████████████░░░░░░░░░░]       │
│  ...                                                  │
└───────────────────────────────────────────────────────┘
```

每个属性：名称 + 当前值/最大值 + 进度条。条色统一灰色（基线数据，只读）。

### 属性步进器 (Story 2 新增)

先天六维和后天属性每个维度显示 [-] [+] 步进按钮，点击 ±1，限制在 0-20（后天）或 1-10（先天）范围内。
通过 `charEditor.updateAttribute()` composable 执行。

### 可变属性 (Story 2 新增)

新增"可变属性"小节，显示体力/精力（当前/上限）+ 声望。
点击打开编辑弹窗，通过 `charEditor.updateVitals()` composable 写入。
体力/精力写入 `{当前, 上限}` 对象。

### 后天六维（当前 1-20）

→ 读 `角色.属性`，schema 驱动字段扫描

```
┌─ 后天六维  当前 · 1-20  [✏] ─────────────────────────┐
│  体质  12 / 20   [██████████████████░░░░░░░░]        │
│  灵力   8 / 20   [████████████░░░░░░░░░░░░░░]        │
│  ...                                                  │
└───────────────────────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| 属性名 | 从 schema `x-display` 扫描，或回退到对象键名 |
| 当前值/最大值 | schema 定义 max；未定义则只显示当前值 |
| 进度条 | 百分比着色：≤25% 红色，≤50% 琥珀色，>50% 绿色 |
| [✏] 按钮 | 打开 SchemaForm Modal 编辑 `角色.属性` |

暂无属性数据时显示 "暂无属性数据"。

```
│  【无锚点状态】                                           │
│  还没有主角锚点。点击下方按钮由 AI 提取。                  │
│  [AI 提取锚点]                ← 提取中显示 loading        │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

| 控件 | 说明 |
|------|------|
| 正面提示词 | `textarea`，编辑锚点 `positivePrompt` |
| 负面提示词 | `textarea`，编辑锚点 `negativePrompt` |
| 启用锚点 toggle | `AgaToggle`，对应 `enabled` |
| 生图默认附加 toggle | `AgaToggle`，对应 `appendByDefault` |
| 场景自动注入 toggle | `AgaToggle`，对应 `autoInjectToScene` |
| [保存锚点] | `AgaButton`，保存到 `系统.扩展.image.characterAnchors` |
| [删除锚点] | `AgaButton danger`，从 anchors 数组移除 player 锚点 |
| [AI 提取锚点] | 仅无锚点时显示。调用 `extractAnchorViaAI`，读取角色基础信息/外貌/身材/衣着生成锚点 |
