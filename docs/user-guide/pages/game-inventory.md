# 背包 (Inventory)

> → [src/ui/components/panels/InventoryPanel.vue](../../../src/ui/components/panels/InventoryPanel.vue)
>
> 游戏内背包面板。路由 `/game`，Tab `inventory`。
> 展示主角持有的物品与货币。
> 物品数据来自 `角色.背包.物品`（`Record<id, Item>`，非数组），
> 货币数据来自 `角色.背包.金钱`（`Record<string, number>` 容器对象，如 `{现金: 100, 金: 50}`）。

```

│                                                                  │
│  [搜索物品…                    ]  [▼ 全部 ]                      │
│  共 12 件物品（全部 15 件）                                       │
│                                                                  │
│  ┌─ 物品卡片 ────────────────────────────────────────────────┐  │
│  │  ⚔  玄铁剑           [已装备]  [稀有]                      │  │
│  │     上古玄铁铸造的长剑，锋利无比                             │  │
```

## 面板头部

→ 模板 `.panel-header`

| 元素 | 说明 |
|------|------|
| 标题 | "背包"，左对齐 |
| 货币栏 | 右侧横向排列。读 `角色.背包.金钱` 容器对象，只展示值 > 0 的币种 |
| [+ 新增物品] | 打开新增物品弹窗（Story 2） |
| [编辑货币] | 打开货币编辑弹窗（Story 2） |

### 货币 Chip

每种货币一个药丸标签：

```
💴 现金 100    🪙 金 50    ⚪ 银 200    🟤 铜 1500
```

| 元素 | 说明 |
|------|------|
| 图标 | 预设映射：现金=💴 / 金=🪙 / 银=⚪ / 铜=🟤 / 其他=💰 |
| 币种名 | `Record` 的 key |
| 数值 | 等宽字体，amber 色加粗 |

无货币或全部为 0 时显示灰色 "无货币"。

## 物品卡片

→ 模板 `.item-card`

每张卡片对应 `Record<id, Item>` 中的一项。

```
┌───────────────────────────────────────────────────────┐
│  ⚔  玄铁剑           [已装备]  [稀有]                  │
│     上古玄铁铸造的长剑，锋利无比                        │
│     武器  ×3                                          │
└───────────────────────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| 类型图标 | 根据 `类型` 字段映射：武器=⚔ / 防具=🛡 / 消耗品=🧪 / 材料=📦 / 任务=📜 / 饰品=💍 / 其他=🔹 |
| 物品名 | `名称` 字段，加粗 |
| [已装备] 徽章 | 仅 `已装备 === true` 时显示，绿色背景 |
| 品质徽章 | 仅 `品质` 字段存在时显示。读 `品质.quality`（对象）或直接读字符串值。颜色：普通=灰 / 优良=sage浅 / 稀有=sage深 / 史诗=amber浅 / 传说=amber深 / 神话=红 |
| 描述 | `描述` 字段，灰色辅助文字，单行截断 |
| 类型标签 | `类型` 字段（如"武器"、"消耗品"），小号灰色 |
| 数量 | `数量` 字段。仅 `数量 > 1` 时显示 `×N`，等宽字体 |
| 已装备左边线 | `已装备 === true` 时卡片左侧有绿色 3px 内阴影指示线 |

### 卡片交互

| 操作 | 行为 |
|------|------|
| 悬停 | 卡片边框高亮（sage 色） |
| 点击 | 选中/取消选中。选中卡片有更深背景色+边框高亮 |
| 选中后 | 底部展开操作栏（slide 动画过渡） |

## 操作栏（选中展开）

→ 模板 `.item-actions`，`Transition name="actions-slide"`

选中物品后在卡片底部滑入显示。点击其他卡片或再次点击当前卡片关闭。

```
┌────────────────────────────────────────┐
│  [使用]  [装备/卸下]  [丢弃]            │
└────────────────────────────────────────┘
```

| 按钮 | 条件 | 行为 | Toast |
|------|------|------|-------|
| [使用] | 始终显示 | 加入操作队列 `use_item` | info: "已将「使用 XX」加入操作队列" |
| [装备] / [卸下] | 仅 `可装备 === true` 时显示 | 加入队列 `equip_item` / `unequip_item` | info: "已将「装备/卸下 XX」加入操作队列" |
| [丢弃] | 始终显示 | 加入队列 `drop_item` | warning: "已将「丢弃 XX」加入操作队列" |
| [✏ 编辑] | 始终显示 | 打开物品编辑弹窗，预填现有数据（Story 2） | — |
| [🗑 删除] | 始终显示 | 弹出确认对话框后直接从状态树移除物品（卡创作级编辑，与"丢弃"不同）（Story 2） | warning: "物品已删除" |

所有操作均写入 `useActionQueueStore`，不直接修改状态树。操作在下次 AI 回合时由引擎消费执行。

按钮悬停颜色：使用=sage / 装备=绿 / 丢弃=红。

## 物品编辑弹窗 (Story 2 新增)

→ 模板 `showItemModal`，宽度 440px

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 名称 | 文本输入 | 是 | 空名时保存按钮禁用 |
| 类型 | 下拉选择 | 否 | 武器/防具/消耗品/材料/任务/饰品/其他 |
| 品质 | 下拉选择 | 否 | 普通/优良/稀有/史诗/传说/神话 |
| 数量 | 数字输入 | 否 | 默认 1，最小 1 |
| 描述 | textarea 2 行 | 否 | — |
| 可装备 | toggle 开关 | 否 | 默认关闭 |
| 已装备 | toggle 开关 | 否 | 仅在可装备=开启时可见。关闭可装备时自动清除 |

新建物品时 ID 自动生成：`${类型}_${时间戳}_${随机码}`。
编辑通过 `useInventoryEditor` composable，不直接调用 setValue。

---

## 货币编辑弹窗 (Story 2 新增)

→ 模板 `showCurrencyModal`，宽度 360px

| 字段 | 类型 | 说明 |
|------|------|------|
| 现金 | 数字输入 | ≥ 0 |
| 铜 | 数字输入 | ≥ 0 |
| 银 | 数字输入 | ≥ 0 |
| 金 | 数字输入 | ≥ 0 |

负值校验：任何币种值为负时保存失败。

---

## 删除确认弹窗 (Story 2 新增)

→ 模板 `showDeleteConfirm`，宽度 380px

显示被删物品名称 + "此操作不可撤销" 警告。
[取消] + [确认删除]。确认后从 `角色.背包.物品` Record 中移除该 key。

货币栏旁的 [⚙ 高级编辑] 按钮跳转到 GameVariablePanel 并导航到 `角色.背包` 路径。
