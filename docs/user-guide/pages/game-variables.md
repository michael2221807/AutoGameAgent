# 游戏变量面板 (Variables)

> → [src/ui/components/panels/GameVariablePanel.vue](../../../src/ui/components/panels/GameVariablePanel.vue)
>
> 游戏内面板。统一数据编辑中心，以树状导航 + 字段列表的双栏布局展示
> 完整游戏状态树，支持搜索、内联编辑、Schema 表单、JSON 编辑器、
> 数据统计和导出。

```
┌─ 游戏变量 ───────── [统计]  [导出 JSON] ─────┐
│                                                │
│  [搜索路径或值…                            ]   │
│                                                │
│  根 / 角色 / 属性                  [{ } JSON]  │
│                                                │
│  ┌─ 树导航 ──────┐  ┌─ 字段列表 ────────────┐ │
│  │                │  │                        │ │
│  │ 📁 基础    ›  │  │  体力   NUMBER   85    │ │
│  │ 📁 技能    ›  │  │  智力   NUMBER   72    │ │
│  │ 📁 装备    ›  │  │  魅力   NUMBER   60    │ │
│  │ ● 等级  15   │  │  名称   STRING   李逍遥 │ │
│  │ ● 经验  340  │  │  存活   BOOLEAN  true   │ │
│  │                │  │  背包   OBJECT   {12}  │ │
│  │                │  │                        │ │
│  └────────────────┘  └────────────────────────┘ │
│                                                │
└────────────────────────────────────────────────┘
```

## 面板头部

| 元素 | 说明 |
|------|------|
| 标题 | "游戏变量"，粗体 |
| [统计] | 幽灵样式按钮（带柱形图标），点击打开统计弹窗 |
| [导出 JSON] | 主色按钮（带下载图标），导出完整状态树为 `game-state-{时间戳}.json` 文件。导出后 Toast 提示 "状态已导出" |

## 搜索栏

> → [src/ui/components/common/SearchInput.vue](../../../src/ui/components/common/SearchInput.vue)

| 元素 | 说明 |
|------|------|
| 输入框 | placeholder "搜索路径或值…"，200ms 防抖 |
| 搜索逻辑 | 将状态树展开为点分路径（最多 6 层深度），按路径匹配关键字，最多返回 100 条结果 |

**搜索激活时：** 主内容区（面包屑 + 双栏）被搜索结果覆盖。

```
┌─ 搜索结果 ──────────────────────────────────┐
│  搜索结果 (23)                               │
│                                               │
│  角色.属性.体力          85                   │
│  角色.属性.体力上限      100                  │
│  世界.状态.天气          晴                   │
│  …                                            │
└──────────────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| 结果头 | "搜索结果 (N)"，大写字母间距 |
| 结果行 | 左侧为点分路径（等宽字体），右侧为值预览（颜色随类型变化） |
| 点击结果 | 自动导航到该路径的父级节点，清空搜索 |
| 空状态 | "没有匹配结果" |

## 面包屑导航

路径导航条，位于搜索栏下方。

```
  根 / 角色 / 属性                      [{ } JSON]
```

| 元素 | 说明 |
|------|------|
| "根" | 始终显示，点击回到状态树顶层 |
| 路径段 | 每段可点击，跳转到对应层级。当前层级（最后一段）为粗体白色 |
| 分隔符 | `/`，灰色半透明 |
| [{ } JSON] | 仅在有选中路径时显示。点击打开 JSON 编辑器弹窗，编辑当前路径下的完整数据 |

## 双栏布局

### 左栏：树导航 (220px 固定宽度)

圆角边框容器，内部可滚动。显示当前路径下的所有子节点。

```
  📁 角色        李逍遥           ›
  📁 世界        5 字段           ›
  📋 NPC列表     3 项             ›
  ●  回合数      15
```

| 元素 | 说明 |
|------|------|
| 图标 | 对象 `📁` 琥珀色文件夹；数组 `📋` sage 色列表；简单类型 `●` 彩色圆点（颜色随类型） |
| 节点名称 | 白色粗体文字 |
| 预览文字 | 等宽字体，半透明。对象/数组显示"N 字段"/"N 项"或首项名称；简单类型显示值 |
| 导航箭头 `›` | 仅对象/数组节点显示。悬停时右移 + sage 色 |

**交互：**

| 节点类型 | 点击行为 |
|----------|----------|
| 对象 / 数组 | 导航进入该节点（更新面包屑，刷新树和字段列表） |
| 简单类型 | 直接进入编辑模式（字符串打开字符串编辑弹窗，数字进入内联编辑） |

**空状态：** "此路径下无数据"。

### 右栏：字段列表 (弹性宽度)

显示当前选中路径下的全部字段。未选中任何路径时显示提示 "选择左侧的分类以查看和编辑字段"。

```
  体力      NUMBER     85            [📋] [↩]
  名称      STRING     "李逍遥"      [📋]
  存活      BOOLEAN    [true]        [📋]
  背包      OBJECT     {12} 铁剑…    [📋]
```

| 元素 | 说明 |
|------|------|
| 字段名 | 白色粗体 |
| 类型标签 | 大写字母，带颜色边框的小药丸（见类型颜色表） |
| 值区域 | 根据类型采用不同编辑方式（见下方详述） |
| 复制按钮 `[📋]` | 悬停行时才显示。复制字段值到剪贴板，Toast 提示 "已复制" |
| 重置按钮 `[↩]` | 仅当值与 Schema 默认值不同时显示。琥珀色，点击需确认 |

**类型颜色表：**

| 类型 | 颜色 |
|------|------|
| string | 绿色 |
| number | 蓝色 |
| boolean | 琥珀色 |
| object | 紫色 |
| array | 粉色 |
| null | 灰色 |

**空状态：** "此路径下无字段"。

## 编辑方式

### 布尔值：点击切换

| 元素 | 说明 |
|------|------|
| 切换按钮 | 显示 `true` 或 `false`。`true` 时绿色背景 + 绿色边框；`false` 时红色文字 + 默认边框 |
| 点击 | 立即取反并写入状态树，Toast "值已更新" |

### 数字：内联输入

| 操作 | 说明 |
|------|------|
| 双击值 | 进入编辑模式，显示 `<input type="number">` |
| 回车 / 失焦 | 提交修改，解析为数字（无效输入归零），Toast "值已更新" |
| Esc | 取消编辑，恢复原值 |

### 字符串：弹窗编辑

| 操作 | 说明 |
|------|------|
| 双击值 | 打开字符串编辑弹窗 |

**字符串编辑弹窗 (560px)：**

```
┌─ 编辑: 名称 ─────────────────────────────────┐
│                                                │
│  ┌────────────────────────────────────────┐    │
│  │                                        │    │
│  │  (多行文本域，8 行高度，CJK 衬线字体)   │    │
│  │                                        │    │
│  └────────────────────────────────────────┘    │
│                                                │
│                          [取消]  [保存]         │
└────────────────────────────────────────────────┘
```

### 复杂对象（有 Schema properties）：Schema 表单弹窗

> → [src/ui/components/editing/SchemaForm.vue](../../../src/ui/components/editing/SchemaForm.vue)

```
┌─ 编辑: 基础属性 ─────────────────────────────┐
│                                                │
│  (SchemaForm 根据 Schema 自动渲染表单字段)     │
│                                                │
│                          [取消]  [保存]         │
└────────────────────────────────────────────────┘
```

| 条件 | 行为 |
|------|------|
| 对象有 Schema `properties` | 打开 SchemaForm 编辑弹窗 (600px) |
| 对象无 Schema `properties`，或为数组 | 导航进入该节点（树视图直接支持数组索引浏览） |

### JSON 编辑器弹窗 (680px)

面包屑栏的 `[{ } JSON]` 按钮打开，对当前路径进行原始 JSON 编辑。

```
┌─ JSON 编辑: 角色.属性 ──────────────────────┐
│                                                │
│  [格式化]          JSON 有效 / JSON 无效       │
│                                                │
│  ┌────────────────────────────────────────┐    │
│  │ {                                      │    │
│  │   "体力": 85,                          │    │
│  │   "智力": 72,                          │    │
│  │   "魅力": 60                           │    │
│  │ }                                      │    │
│  └────────────────────────────────────────┘    │
│                                                │
│  ⚠ Unexpected token … ← JSON 语法错误时显示   │
│                                                │
│                          [取消]  [保存]         │
└────────────────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| [格式化] | 将当前文本解析并重新缩进为 2 空格格式 |
| 状态徽章 | 实时校验。有效时绿色 "JSON 有效"；无效时红色 "JSON 无效" |
| 文本域 | 等宽字体，320px 最小高度，可拖拽调整大小 |
| 错误消息 | JSON 解析失败时在文本域下方显示红色错误详情 |
| [保存] | JSON 无效时灰化禁用。保存后 Toast "JSON 已保存" |

## 统计弹窗 (360px)

```
┌─ 状态树统计 ─────────────────────────────────┐
│                                                │
│  估算大小          128.3 KB                    │
│  叶字段总数        1,247                       │
│  最大深度          8 层                        │
│  根节点数          6                           │
│                                                │
│                   [关闭]  [导出 JSON]           │
└────────────────────────────────────────────────┘
```

| 统计项 | 说明 |
|--------|------|
| 估算大小 | 状态树 JSON 序列化后的字符数 / 1024，保留 1 位小数，单位 KB |
| 叶字段总数 | 递归计算所有非容器（非 object/array）节点数 |
| 最大深度 | 状态树最深嵌套层数 |
| 根节点数 | 顶层键的数量 |

页脚两个按钮：[关闭] 关闭弹窗；[导出 JSON] 触发导出（同头部的导出按钮）。

## 重置确认弹窗 (400px)

当字段值与 Schema 默认值不同时，字段行末尾显示琥珀色 `[↩]` 按钮。点击后弹出确认：

```
┌─ 确认重置 ───────────────────────────────────┐
│                                                │
│  确定要将 角色.属性.体力 重置为默认值吗？      │
│  此操作不可撤销。                              │
│                                                │
│                       [取消]  [确认重置]        │
└────────────────────────────────────────────────┘
```

[确认重置] 为红色危险样式。确认后写入默认值，Toast "已重置为默认值"。

## 未加载状态

游戏数据未加载时，面板整体显示居中文字 "尚未加载游戏数据"。

## 深链接支持 (Story 2 新增)

GameVariablePanel 支持通过 URL 查询参数从其他面板跳转定位：

| 参数 | 说明 |
|------|------|
| `?path=角色.基础信息` | 自动导航到指定路径 |
| `?from=character` | 显示"← 返回 [来源面板]"链接 |

各面板的 [⚙ 高级编辑] 按钮使用此机制跳转到 GameVariablePanel 并深链接到对应数据路径。
